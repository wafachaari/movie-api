Documentation

Request URL HTTP method Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET NONE A JSON object holding data about all the movies
Return data about a single movie by title to the user /movies/[title] GET NONE A JSON object holding data about a single movie
Return data about a genre by Name /movies/genre/[Name] GET NONE A text message indicating the description of a specific genre
Return data about a director by name /movies/director/[name] GET NONE A text message indicating director's inormation
Allow new users to register /users POST JSON A JSON object holding data about the user to add, structured like: { name : "Paulina Gomez", Email : ....@gmail.com, favorite-movies: {} } }
Allow users to update their user info (username) /users/[username] PUT JSON a json object containing info of the user
Allow users to add a movie to their list of favorites /users/[username]/movies/[favorite-movies] Put None a json object containing user information
Allow users to remove a movie from their list of favorites /users/[username]/[movies] DELETE NONE A text message indicating whether the movie was successfully removed
Allow existing users to deregister /users/[username] DELETE NONE text that a user email has been removed
Return a list of actors /actors GET NONE Json object containing a list if all actors
